No CSS, possuimos os seletores por ID

  • Seletor por Tipo: Busca elementos por uma tag HTML
  • Seletor por ID Busca elementos através do atributo "id"
  • Seletor por Classe Busca elementos através do atributo "class"
  • Seletor UniversalSeleciona todos os elementos HTMl
  • Seletor de AtributoSeleciona elementos que possuem um atributo específico em nosso documento HTMl e conseguimos também buscar atributos com um valor específico
